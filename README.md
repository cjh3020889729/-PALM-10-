# 飞桨常规赛：PALM病理性近视预测 第10名方案

本方案使用TNT模型进行训练与预测，在有限训练次数下，取得了较稳定的成绩：0.99515

**采取的训练图像预处理方案：**

* 随机垂直翻转
* 随机角度翻转--0~20度
* 缩放大小--（520，520）
* 归一化--mean:[0.2, 0.3, 0.5], std:[0., 0., 0.]

**采取的验证与预测图像预处理方案：**

* 缩放大小--（520，520）
* 归一化--mean:[0.2, 0.3, 0.5], std:[0., 0., 0.]

**数据集划分比例：**0.8

**TNT模型比较：**

* 微调参数： tnt_s_patch16_224 得分 > tnt_b_patch16_224 得分

**后期优化方向：**

* 更合适的处理方式
* patch大小与数量
* 图片输入大小等

## 环境说明

* PaddlePaddle 2.0.2
* opencv > 3.2
